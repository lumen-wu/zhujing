## Basic concepts

From homotopytypetheory.org:
- **Propositional equality = homotopy** — two terms are equal if there's a path (continuous transformation) between them
- **Type isomorphism = homotopy equivalence** — two types are "the same" if they're homotopy equivalent
- **Univalence axiom** — equality on the universe = homotopy equivalence of small types

Key idea: two proofs that can be continuously deformed into each other are considered "the same proof."

## Connection to 阿哲's speculation

If cognition has homotopy-like structure:
- Two thought-processes that reach the same understanding via different paths might be "homotopic"
- Understanding (η) might be detecting when you're in the same equivalence class
- S_E would be the invariant structure preserved across all paths
